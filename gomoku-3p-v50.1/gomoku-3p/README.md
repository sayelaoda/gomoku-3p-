# 🎮 三人五子棋联机对战平台

一个简单好用的三人五子棋联机对战网站，支持朋友之间在线对战。

## ✨ 功能特点

- **三人对战** - 黑棋、白棋、红棋轮流下棋
- **房间系统** - 创建房间，邀请朋友加入
- **实时通讯** - 游戏内实时聊天
- **响应式设计** - 支持电脑和手机浏览器
- **获胜判定** - 自动检测五子连珠

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务器

```bash
npm start
```

服务器默认运行在 `http://localhost:3000`

### 3. 开始游戏

1. 打开浏览器访问 `http://localhost:3000`
2. 输入你的昵称
3. 点击"创建房间"获得房间号
4. 把房间号分享给朋友
5. 朋友输入房间号加入
6. 至少2人后点击"开始游戏"

## 📱 使用说明

### 创建房间
- 输入你的名字
- 点击"创建房间"
- 分享房间号给朋友

### 加入房间
- 输入房间号（6位字母）
- 输入你的名字
- 点击加入

### 游戏规则
- 三人轮流下棋：黑棋 → 白棋 → 红棋
- 最先连成5子（横/竖/斜）获胜
- 每局结束后可重新开始

## 🛠️ 技术栈

- **前端**: HTML5 + CSS3 + 原生JavaScript
- **后端**: Node.js + Express
- **实时通信**: WebSocket
