# 🎮 多人五子棋联机对战平台

一个简单好用的多人五子棋联机对战网站，支持2-10人同时在线对战。

## ✨ 功能特点

- **多人对战** - 支持2-10人同时对战
- **丰富颜色** - 10种棋子颜色可选
- **吃子规则** - 周围无气的棋子被提掉
- **房间系统** - 创建房间，邀请朋友加入
- **实时通讯** - 游戏内实时聊天
- **响应式设计** - 电脑和手机浏览器完美体验
- **掉线重连** - 游戏内离线自动检测，支持重连
- **房主权限** - 房主离开房间自动解散
- **落子提示** - 最新落子高亮显示

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务器

```bash
npm start
```

服务器默认运行在 `http://localhost:3000`

### 3. 开始游戏

1. 打开浏览器访问 `http://localhost:3000`
2. 输入你的昵称
3. 点击"创建房间"获得房间号
4. 把房间号分享给朋友
5. 朋友输入房间号加入
6. 选择喜欢的棋子颜色
7. 至少2人后点击"开始游戏"

## 📱 使用说明

### 创建房间
- 输入你的名字
- 点击"创建房间"
- 分享房间号给朋友

### 加入房间
- 输入房间号（6位字母）
- 输入你的名字
- 点击加入

### 游戏规则
- 多人轮流下棋
- 最先连成5子（横/竖/斜）获胜
- 吃子规则：周围无气的棋子被提掉
- 每局结束后可重新开始

## 🎯 游戏特色

### 智能落子
- 自动检测连珠获胜
- 提子后更新得分
- 最新落子高亮提示

### 断线重连
- 游戏内掉线不会立即出局
- 重连后恢复游戏状态
- 房主可选择等待或移除离线玩家

### 房间管理
- 房主可解散房间
- 其他玩家离开不影响游戏
- 房间列表实时更新

## 🛠️ 技术栈

- **前端**: HTML5 + CSS3 + 原生JavaScript
- **后端**: Node.js + Express
- **实时通信**: WebSocket (ws)
